# Changelog

All notable changes to the Private AI File Organiser will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-31

### Added
- Initial release of Private AI File Organiser
- Core file classification system (hybrid rule-based + AI)
- Integration with Ollama for local AI inference
- Folder watching and automatic file detection
- File action manager (move, rename, delete, archive)
- Duplicate file detection and cleanup
- SQLite-based local database for tracking
- Web-based dashboard with FastAPI
- License validation system (200-key limited release)
- Time saved tracking and statistics
- Dry run mode for safe testing
- Undo functionality for recent actions
- Comprehensive logging system
- CLI interface with multiple commands
- Configuration management via JSON
- Documentation and README
